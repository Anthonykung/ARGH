Detailed Description

The Buffer class represents the buffer module in the GenTL module hierarchy.

This Buffer class contains a method that can be used to temporarily hold a chuck of memory from the acquisition engine. It also contains methods that can retrieve information of the received data (held data) such as pointer to an image, image size, and pixel format.
