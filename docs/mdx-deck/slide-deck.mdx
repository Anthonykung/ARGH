# Aerospace Recorder
# for Graphical History

The ARGH Project is to create a camera system for a large commercial and/or military drone to record graphical data or images using industrial vision cameras.

---

# ARGH Project System

1. Nvidia Jetson AGX Xavier
2. GigE Vision Camera
3. USB Vision Camera
4. PoE Switch
5. VICOR Power Supply
6. Pushbutton
7. Waveshare e-Paper Display
